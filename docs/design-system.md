# OrderUp Design System

## Brand Colours
| Name | Hex | Usage |
|------|-----|-------|
| Brand Red | `#C0474A` | Primary buttons, headers, active states |
| Brand Gradient | `#C0474A → #E8726A` | Hero banners, CTAs |
| Background | `#F7F5F2` | Page backgrounds |
| Card White | `#FFFFFF` | Card surfaces |
| Card Tint | `#FFF8F6` | Selected/highlighted cards |
| Text Primary | `#1a1a2e` | Headings, body text |
| Text Muted | `#888888` | Subtitles, placeholders |
| Border | `#EBEBEB` | Card borders, dividers |

## Typography
- Font: System UI / sans-serif
- Heading: 700–800 weight
- Body: 400 weight, 1.6 line-height
- Small/labels: 0.75–0.85rem

## Spacing
- Page padding: 16px
- Card padding: 1.25rem–2rem
- Card border radius: 14px–18px
- Button border radius: 2rem (pill)

## Component Styles
### Buttons
- Primary: gradient `#C0474A → #E8726A`, white text, pill shape
- Disabled: `#E0E0E0` background, `#aaa` text

### Cards
- Background: white
- Border: 1px solid `#EBEBEB`
- Shadow: `0 2px 12px rgba(0,0,0,0.07)`
- Hover: lift with stronger shadow

### Navigation
- Background: `#C0474A`
- Logo box: white, 10px radius
- Icons: white, 34px circular background at 20% opacity

## Icons
- Library: Lucide React
- Stroke width: 2
- Size: 16px (nav), 20px (buttons), 24px (cards)
